Part1: Fetching data from Wikipedia:¶

The wikipedia article "List of mountains by elevation" is organized into several tables of data:¶

- one table for mountain that are atleast 8,000m in height,¶

- Another table for 7,000m mountains (7,000m - 7,999m),¶

- Another table for 6,000m mountains (6,000m - 6,999m),¶

... and also so on...¶

The pd.read_html(...) function in the pandas library is designed to read the data from the table found in webpages.¶

In [5]:
#Import the libraries:
import pandas as pd
import requests
In [43]:
url = "https://en.wikipedia.org/wiki/List_of_mountains_by_elevation"

# Add headers to mimic a real browser
headers = {"User-Agent": "Mozilla/5.0"}
response = requests.get(url, headers=headers)

# Now pass the content to pandas
dfList = pd.read_html(response.text)
